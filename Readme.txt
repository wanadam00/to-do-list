1. Open laragon (localhost) terminal or terminal that compitable.
2. Run command 'cd C:\laragon\www\'
3. Run command 'git clone https://github.com/wanadam00/to-do-list.git'
4. Run command 'compser install'
5. Run command 'npm install'
6. Run command 'php artisan key generate'
7. Run command 'php artisan migrate:fresh'
8. Run command 'cp .env.example .env'
9. Run command 'code .'
10. restart localhost to enable ssh certificate
11. Run command 'npm run dev' in terminal
